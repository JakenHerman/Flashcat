Get started with flashcat.

The idea of flashcat is to be a simpler, automatic application installer and updater for Macintosh and Windows operating systems. In the future, I will look to branch out into linux systems as well, but for now, we'll keep it small. As flashcat is right now, it is not functional. If you'd like to hijack the code and make it your own, feel free to clone: 

$ cd your_directory
$ git clone https://github.com/JakenHerman/Flashcat.git
